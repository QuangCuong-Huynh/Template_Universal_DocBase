# Project Charter

**Domain:** `PM` &nbsp; **Document ID Pattern:** `PM-[PROJECT]-CHARTER-[DATE]-01`

> Formally authorizes the project, grants the PM organizational authority, and defines scope, stakeholders, budget, milestones, and high-level risks.
